3.MealViewController,进行编辑的界面，还有详情的界面，可以点击保存和更新一个新的table item
4.MealTableViewCell, 的tableview的一个secletion，里面包含三个界面，UIImage, lable,星级
5.Meal，数据的存储类，保存有食品的信息

6、FoodTrackerTests测试类，主要测试数据的逻辑
